Overview

This web app is designed to examine the transfer function of the CPMA. The app uses data derived from tandem CPMA (TCPMA) measurements conducted by Naseri et al. (2023).
A triangular transfer function was used to approximate the CPMA transfer function and deviations from the idealized triangular transfer function were determined by two factors: (i) the width factor (Œº), and (ii) the height factor (Œª), such that a value of 1 for each factor implied no deviation between the actual and theoretical triangular CPMA transfer functions.

The width factor higher than one (Œº>1), indicating that the CPMA transfer function is narrower than the idealized triangular transfer function.

These two factors are calculated from multivariate nonlinear functions fitted to TCMPA measurements by indicating either CPMA mass set point (mp), CPMA resolution (Rm), and CPMA flow rate (Q) or CPMA rotational speed (œâ) and voltage (V). In case that the CPMA flow rate is set to 0.3, 1.5, 4 or 8 LPM, the loss factor is calculated directly from fitting function derived for the corresponding CPMA flow rate, whereas for other flow rates, the loss factor is obtained from simple interpolation between the closest lower and higher CPMA flow rates.
